# 🥗 NUTRIFIT

**NUTRIFIT** is a healthcare app designed to promote healthy living by providing users with comprehensive diet plans, strategic workout routines for all phases, and home remedies for common ailments.

## ✨ Features
- **Diet Plans**: Extensive selection of healthy diet options.
- **Workout Strategies**: Tailored workouts for various fitness levels.
- **Home Remedies**: Effective solutions for common diseases.

## 🛠️ Technologies Used
- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Authentication**: JWT for secure user authentication
- **API Integration**: For external services
